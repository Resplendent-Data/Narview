# Narview Mobile

Flutter client for Narview's mobile Pull Request review workflow.

Android is the first acceptance target. iOS project files are kept present so the
client can be brought up on iPhone and iPad once the Android review loop is stable.

## Commands

From the repository root:

```bash
npm run mobile:analyze
npm run mobile:test
```

From this directory:

```bash
flutter run
flutter build apk --debug
```
