# Use Tauri, React, TypeScript, and Rust

Narview is built as a Tauri v2 desktop app with a React and TypeScript frontend backed by a Rust core. This gives Narview mature web UI primitives for complex review surfaces and a native-capable Rust backend for GitHub integration, caching, secure storage, and performance-sensitive indexing, while avoiding Electron's larger bundled Chromium runtime and Flutter's weaker fit for code-review-specific UI infrastructure.
